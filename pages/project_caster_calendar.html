Discord Event Calendar


  I developed a Discord-integrated event management system in Python that streamlines event organization and communication for Discord communities. I did it using Claude CLI mainly.

Event Management & Discord Integration


  Organizers can create, edit, and delete events through a web interface. The system automatically notifies relevant Discord users via pings when events are created, modified, or deleted.

  A dedicated Discord channel displays an auto-generated image summarizing all upcoming events, keeping the community informed in one centralized location.

Administration Panel


  The admin panel provides complete control over Discord server configuration, including guild management, user assignments, and team organization.

  Administrators can configure event channels, image channels, and manage which Discord users receive event notifications.

Google Calendar Integration

  Users can sync events with their personal Google Calendar through a generated feed URL. This allows team members to view upcoming events in their preferred calendar application, ensuring they never miss an important event.



Technical Implementation

  •     • Built with Python Flask for the web backend
  •     • Discord.py library for Discord bot integration
  •     • Created a custom Discord App with proper OAuth2 permissions
  •     • Automated image generation using Python PIL for event summaries
  •     • Real-time Discord webhook integration for instant notifications
  •     • iCalendar feed generation for Google Calendar synchronization
  •     • SQLite database for event and user data persistence
  •     • Self-hosted on a VPS for full control and privacy

Key Features

  •     • Automatic Discord pings on event creation, updates, and deletions
  •     • Visual event summary images posted to Discord channels
  •     • Multi-guild support with separate configurations
  •     • Role-based access control (organizers vs. regular users)
  •     • Team management system for organizing participants
  •     • Calendar export for external calendar applications

  The application was thoroughly tested on a personal Discord server to ensure reliability and proper permission handling before deployment.
