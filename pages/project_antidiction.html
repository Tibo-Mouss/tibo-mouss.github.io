Antidiction


  I developed Antidiction, an Android application designed to help people fighting addiction by tracking their daily progress and victories. Built with Kotlin in Android Studio, the app provides comprehensive tools for recovery management and motivation.
  I did it using Claude CLI mainly.

Daily Tracking & Calendar System


  The main screen features an interactive calendar where users can track their daily progress. For each day, users can record multiple metrics:

  • Craving Level: Rate addiction cravings from 0 (no cravings) to 5 (intense cravings)
  • Day Quality Rating: Evaluate overall day quality on a scale from 0 (bad day) to 5 (great day)
  • Success/Failure State: Mark whether they succeeded or faced challenges (displayed as green for positive days, purple for negative ones)
  • Sports Activity: Track physical exercise engagement

  The calendar displays all tracked information at a glance using color-coded cells and emoji indicators, providing immediate visual feedback on progress and patterns.

Progress Statistics & Analytics


  The statistics page provides detailed insights into recovery progress through interactive charts and metrics.

  Users can select different time periods (week, month, year) and compare their current performance with previous periods. The app displays average day ratings, craving levels, and visual trend graphs that help identify patterns and progress over time.

Personal Diary with Speech-to-Text


  The diary feature allows users to document their thoughts, feelings, and experiences throughout their recovery journey.

  Users can write entries manually or use the integrated speech-to-text functionality for easier and faster journaling. This feature helps users reflect on their progress, identify triggers, and maintain a personal record of their recovery process.


Goal Setting & Reward System

  The reward screen features a visual recovery journey where users can set weekly milestones and associate rewards with each achievement. Users define custom goals and rewards (represented by emojis), creating a gamified progression system that maintains motivation throughout the recovery process. The journey is displayed as a colorful roadmap with nodes representing each week's milestone.
