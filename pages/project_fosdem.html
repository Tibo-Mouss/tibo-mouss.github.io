Fosdem Convention


  FOSDEM is a non-commercial event where people meet to discuss and present tools / ideas about Open Source technologies.
  I was there at the 2026 edition in Brussels, and got to discuss with professionals of the sector, and listen to some very interesting talks.



Best Talks I've attended

🥉An Efficient Git Workflow For High-Stakes Projects

  Very nice and smart approach to git usage. Vladislav Shpilevoy shows the small details we can add to our daily work routines to drastically improve group work efficiency.

🥈How to Make Package Managers Scream

  Here Kenneth Hoste shows with a lot of humour and sarcasm all the mistake one can make (or avoid to make !) when creating an open source project.

🥇How to instrument go without changing a single line of code

  Kemal Akkoyun and Hannah Kim show solutions to add metrics to a project without changing the source code, by injecting the code at compilation for example.
